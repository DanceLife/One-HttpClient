What's in here

This is a basic implementation of a navigation bar using bootstrap 4.

How to use it

Clicking the option on the navigation bar to Register will allow a visiting user to create a new account into the backend.

You will need to create a Firebase project and provide the apiKey and authDomain to see the functionality on place.

When testing locally you can visit the Profile page to enter the Firebase apiKey and authDomain. That information will be save in your device local storage so you can save your code in a public repository without compromising your access credentials.

For production implementation you want to add those 2 pieces of informatin into the app.component.ts to make the functionality accessible to visiting users.
