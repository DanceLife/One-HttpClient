What's in here

This one is a basic implementation of a search box. Angular solutions are single page applications or SPAs. SPAs and are not directly used in a context where search engine indexing is either a requirement or desirable. If you want your Angular solution to be parsed by a search engine you will have to ensure a template plus some precomposed content gets shipped by your server for all valid URLs in your solution. How you do this will vary depending on your server architecture.
