What's in here

Here we are looking to the new Angular 6 HttpClient. No that there is anything wrong with the old HTTP client, but the new client packs a set of features that make it attractive over the old one. Features like interceptors and the capability to define the datatype to be received, with the body directly returned instead of the whole response.

We are going to be saving and retrieving information with our requests hence is recomended that you have your environment ready. If you are running this locally you will need to run the following line in your console to install some dependencies, then go to Register/Environment and follow the instructions


npm install firebase angularfire2 @types/firebase


If you want to test the live version then just go to Register/Environment and follow the instructions.
